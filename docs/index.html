品質・制作方針

Webサイト制作において意識している品質や実装方針を紹介しています。

保守性を重視したCSS設計

BEMをベースにクラス設計を行い、コンポーネント単位で管理しています。
SCSSを用いた分割構造により、改修時の影響範囲を最小限に抑えることを意識しています。

画像最適化

表示速度と画質のバランスを考慮し、画像の形式・容量を適切に選定して実装します。

  • 画像の用途に応じて PNG / JPEG / SVG など適切なフォーマットを使用
  • 必要に応じて WebP / AVIF を利用し、ファイルサイズを削減
  • Lazy loading を使用し、初期表示のパフォーマンスを改善
  • width / height を指定しCLS（レイアウトシフト）を防止
アクセシビリティ・ユーザビリティ

ユーザー環境に関わらず利用しやすいWebサイトを目指し、アクセシビリティとユーザビリティに配慮した実装を行います。

  • セマンティックHTMLを意識したマークアップ
  • 見出し構造（h1〜h6）を適切に設計
  • 画像には適切な alt 属性を設定
  • キーボード操作に配慮した実装
  • スマートフォンで操作しやすいタップ領域を確保
